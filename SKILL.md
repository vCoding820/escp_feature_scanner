# ESCP_FEATURE_SCANNER — 特性画像自动生成

扫描嵌入式控制器底层软件（C 语言）代码工程，固定生成 21 个硬件特性的 YAML 画像文件。

## When to Activate

- 用户提供一个 C 代码工程路径，要求扫描生成特性画像
- 用户要求生成知识基线 / 特性画像
- 用户要求分析代码中涉及的硬件特性
- 用户要求更新已有的特性画像

## 核心约束

1. **全自动执行** — 收到项目路径后自动完成扫描和生成
2. **固定 21 个特性** — 无论代码是否涉及，都生成全部 21 个特性的画像
3. **YAML 固定规格** — 每个画像严格遵循 `knowledge-profile-schema.yaml`
4. **只读扫描** — 绝不修改源代码文件
